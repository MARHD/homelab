## Homelab
- My Personal Homelab setup files.

# How To Use
- Type "apt install git"
- Type "git clone https://github.com/MARHD/homelab.git"
- Type "cd homelab"
- Create a .env file based on the env.example file, using touch or nano etc.
- Type "chmod +x setup.sh && ./setup.sh" and let it handle the rest.
